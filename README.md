# learning-AmazonDocumentDB

👓👓👓 Amazon DocumentDBを学ぶためのリポジトリ。  

## 環境情報

| Name | Version |
| --- | --- |
| Terraform | v1.3.7 |
| AWS CLI | 2.9.17 |

## 環境構築

Terraformを使用してAWS上にリソースを構築する。  

```shell
terraform init
terraform apply
```

## イロイロ情報共有

### データベースクライアント

DynamoDB用のNoSQL Workbenchは[こちら](https://docs.aws.amazon.com/ja_jp/amazondynamodb/latest/developerguide/workbench.html)からダウンロード可能。
